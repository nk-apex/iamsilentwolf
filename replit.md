# SilentWolf Portfolio

## Overview
A cyberpunk-themed tech explorer portfolio website showcasing projects built by SilentWolf. Features neon green aesthetic, glassmorphism cards, grid background pattern, and futuristic typography. Identity: "I am just an explorer — tech is my hobby."

## Recent Changes
- Initial build: Portfolio website with Navbar, Hero, Projects grid, Footer, Chat widget
- Cyberpunk theme with Orbitron + JetBrains Mono fonts
- Neon green (#39FF14) color palette on deep black background
- Replaced favicon with wolf image, also used as OG thumbnail
- Added WhatsApp shortcuts (group, channel, phone numbers) to navbar and footer
- Toned down "Digital Weapons" neon glow
- Changed identity from "developer" to "tech explorer" throughout

## Architecture
- Frontend-only portfolio (no database needed)
- Single page app with wouter routing
- Components: navbar, hero, projects, footer, chat-widget
- Custom CSS classes for neon effects (neon-text, neon-btn, glass-card, etc.)
- Wolf favicon also serves as OG/Twitter thumbnail image

## User Preferences
- Dark-first cyberpunk design
- Neon green on black with glassmorphism
- Orbitron for headings, JetBrains Mono for body
- Grid background pattern
- Identity: tech explorer, not developer ("tech is my hobby")
- "Digital Weapons" text should not be too luminous
