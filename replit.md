# SilentWolf Portfolio

## Overview
A cyberpunk-themed developer portfolio website showcasing projects built by SilentWolf. Features neon green aesthetic, glassmorphism cards, grid background pattern, and futuristic typography.

## Recent Changes
- Initial build: Portfolio website with Navbar, Hero, Projects grid, Footer, Chat widget
- Cyberpunk theme with Orbitron + JetBrains Mono fonts
- Neon green (#39FF14) color palette on deep black background

## Architecture
- Frontend-only portfolio (no database needed)
- Single page app with wouter routing
- Components: navbar, hero, projects, footer, chat-widget
- Custom CSS classes for neon effects (neon-text, neon-btn, glass-card, etc.)

## User Preferences
- Dark-first cyberpunk design
- Neon green on black with glassmorphism
- Orbitron for headings, JetBrains Mono for body
- Grid background pattern
